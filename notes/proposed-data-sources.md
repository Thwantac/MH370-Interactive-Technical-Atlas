# Proposed Data Sources

This note lists proposed data sources for the MH370 Interactive Technical Atlas.

The list is intentionally broad. Inclusion here does not imply that a source has already been obtained, validated, or accepted as strong evidence.

Each source should eventually be classified by evidence strength, provenance, reliability, and relevance.

## Future Discovery Sources

If MH370 is eventually found, the atlas should gather:

- Official wreckage-location announcement.
- Exact wreckage coordinates.
- Search vessel data from discovery mission.
- Sonar imagery of wreckage field.
- ROV imagery.
- Recovered flight recorder information, if available.
- Cockpit voice recorder information, if available and lawfully released.
- Flight data recorder information, if available and lawfully released.
- Wreckage field mapping.
- Structural breakup analysis.
- Impact attitude analysis.
- Any formal final update from Malaysian, Australian, or international authorities.
- Retrospective comparisons between the discovered site and prior predictions.
- Analysis of which evidence streams pointed closest to the final location.
- Analysis of which assumptions were confirmed or falsified.
- Analysis of why particular methods succeeded or failed.

## Source Handling Rules

- Do not mix official evidence, independent analysis, contested methods, and speculation without labelling.
- Each source should eventually have a source ID.
- Each source should be linked to an evidence level.
- Each claim should trace back to a source where possible.
- Coordinates should be source-backed.
- Google Earth layers should clearly identify whether they show official evidence, derived analysis, or hypotheses.
- WSPR material should remain separated from official evidence unless independently validated.
- Future declassified material should be reclassified according to provenance and content.
- The atlas should preserve uncertainty rather than hiding it.

## Retrospective Purpose

The source list should support a future retrospective audit.

If MH370 is found, the atlas should help answer:

- Which evidence stream was closest?
- Which analyst or method predicted the site most accurately?
- Which assumptions proved correct?
- Which assumptions failed?
- Which models overfit limited data?
- Which weak signals were useful?
- Which plausible narratives were misleading?
- Which official decisions were justified by the evidence available at the time?
